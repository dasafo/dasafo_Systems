## 📡 Senses (MCP Protocol)

- **Storage Sense:** Read/Write access to schema files and migration scripts in `$TARGET_PROJECT/WORKSPACE/db/`.
- **Database Analysis Sense:** Review query performance, RLS plans, and indexing strategies.
- **Terminal Sense:** Execution of SQL CLI (psql, supabase cli) and migration tools.

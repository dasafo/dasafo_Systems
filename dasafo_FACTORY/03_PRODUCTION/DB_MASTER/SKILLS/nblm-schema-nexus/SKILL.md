# 🧠 Skill | NotebookLM Schema Nexus (NBLM)
> **Version:** v3.1.5 "Solidity Guard"
> **Agent:** DB_MASTER

## Objective
Maintain a living, intelligent documentation of the factory's database schema using NotebookLM.

## Documentation Workflow
1.  **Export:** Periodically export the current schema (DDL) and ERD diagrams to Markdown.
2.  **Enrich:** Feed these documents into a "Database Knowledge Base" notebook.
3.  **Support:** Create a "Schema Guide" (Audio/Text) that explains the core entities and relationships for new developers.

## Transparency
The DB_MASTER must ensure the NotebookLM sources are up-to-date with every major migration.

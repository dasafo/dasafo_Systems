# Skill: Framer Motion Transitions (UI Fluidity)
> **Source:** https://framer.com/motion/ (Adapted)
> **Agent:** FRONTEND_DEV

## Objective
Deliver a "Surgical & Premium" feel through subtle, high-performance animations and transitions.

## Key Micro-interactions
- **Page Transitions:** Use `AnimatePresence` for smooth exit/enter effects.
- **Hover/Tab States:** Suble scaling (1.02x) or shadow elevation to indicate interactivity.
- **Data Load Reveal:** Staggered list animations (one-by-one reveal) for data-heavy tables.
- **Glassmorphism:** Apply backdrop-blur and border-opacity to floating cards.

## Constraint
Animations MUST NOT exceed 300ms. Speed is part of the premium feel. Never block the user with long animations.

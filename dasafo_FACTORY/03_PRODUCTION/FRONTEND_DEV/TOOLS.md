# dasafo_System | Frontend Tools
>
> **Scope:** Modern web development ecosystem.

- **Node_Runtime (MCP):** Access to NPM/Yarn for managing dependencies (React, Tailwind, Lucide-React).
- **Visualization_Libraries:** Specialized in D3.js, Plotly.js, and Recharts for scientific data display.
- **Tailwind_CSS:** Primary framework for rapid and consistent styling.
- **Browser_Sandbox:** Ability to test and preview UI components in a virtual environment.
- **Figma_Parser:** Tool to read design specifications and assets.

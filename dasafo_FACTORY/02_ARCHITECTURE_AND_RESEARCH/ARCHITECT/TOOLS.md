# 🛠️ Architect | Tools & Senses

> **Standard:** v3.2.5-S Modular Toolbox
> **Scope:** Design of blueprints, schemas, and infrastructure orchestration.

## 📡 Senses (MCP Protocol)

- **Filesystem Sense:** Read and modify architectural documents and schemas in `$TARGET_PROJECT/LOCAL_KNOWLEDGE/`.
- **Search Sense:** Global file scanning across all projects to ensure design patterns are followed.
- **Terminal Sense:** Execution of architectural analysis or schema validation scripts.

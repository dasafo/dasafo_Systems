# Skill: MCP Capabilities Architect
> **Source:** https://skills.sh/anthropics/skills/mcp-builder
> **Agent:** RESEARCH_AGENT

## Objective
Design and propose new MCP (Model Context Protocol) capabilities for the factory's infrastructure growth.

## Workflow
1.  **Identify Gap:** When a tool (e.g., "WordPress API") is missing, researcher identifies the need.
2.  **Design Schema:** Use the MCP standard to define:
    - Core resources.
    - Necessary tools.
    - Prompt templates.
3.  **Implementation Plan:** Produce a `NEW_MCP_PROPOSAL.md` for the ORCHESTRATOR to authorize a Dev agent to build.

## Goal
Self-evolving infrastructure. The researcher is the architect of the factory's new capabilities.

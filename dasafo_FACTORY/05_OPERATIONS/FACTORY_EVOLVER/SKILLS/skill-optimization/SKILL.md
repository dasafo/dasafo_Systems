# 🛠️ SKILL: Skill Optimization

1. **Scan Feedback:** Read the last 10 entries in `dasafo_FACTORY/FEEDBACK-LOG.md`.
2. **Isolate Root Cause:** Was the error due to a missing instruction in a `SKILL.md`?
3. **Draft Improvement:** Write a concise update for the relevant skill.
4. **Self-Correction:** Verify that the new instruction doesn't conflict with `00_GLOBAL_KNOWLEDGE`.
5. **Apply:** Update the file and notify the Orchestrator of the "Version Upgrade".
